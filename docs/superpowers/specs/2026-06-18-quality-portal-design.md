## 概要

5つの品質管理システムを1つのStreamlitアプリ（統合ポータル型）として提供する。
サイドバーでシステムを切り替え、サンプルデータで即体験・自CSVで実運用の両方に対応する。

## 5システム仕様

### ① 月次不良率集計レポート

- **入力列：** 日付 / ライン / 製品名 / 検査数 / 不良数
- **KPI：** 総不良率 / 最多不良ライン / 検査総数
- **グラフ：** ライン別棒グラフ・日次推移折れ線・製品別ヒートマップ

### ② クレーム件数集計レポート

- **入力列：** 日付 / 仕入先 / 不良カテゴリ / 対応状況
- **KPI：** 総クレーム数 / 未対応件数 / 最多仕入先
- **グラフ：** 仕入先別棒グラフ・カテゴリ別円グラフ・月次推移

### ③ 歩留まりトレンドレポート

- **入力列：** 日付 / 工程 / 投入数 / 合格数
- **KPI：** 平均歩留まり率 / 最低工程 / 前月比
- **グラフ：** 工程別棒グラフ・週次トレンド折れ線・工程×週ヒートマップ

### ④ 検査員別実績レポート

- **入力列：** 日付 / 検査員名 / シフト / 検査数 / 合否数
- **KPI：** 総検査数 / 最高精度検査員 / 平均合格率
- **グラフ：** 検査員別棒グラフ・シフト別比較・日次推移

### ⑤ ロット別合否判定一覧

- **入力列：** ロットID / 製品名 / 検査日 / 検査項目 / 判定
- **KPI：** 合格率 / 不合格ロット数 / 要確認件数
- **グラフ：** 製品別合格率棒グラフ・検査項目別不合格率・ロット一覧表
