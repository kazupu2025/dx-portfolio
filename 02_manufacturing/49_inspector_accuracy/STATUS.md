# C-103 検査員別 検査数・不良検出率・精度レポート

- **status**: production-ready
- **industry**: 製造業
- **difficulty**: ★★★
- **path**: 02_manufacturing/49_inspector_accuracy/

## 概要

検査員別の検査数、検出した不良数、実際の不良数から、各検査員の検査精度（不良検出率）を分析します。

### 主要機能

- **KPI表示**: 平均検出率、最低検出員、見逃し総数、総合判定
- **棒グラフ**: 検査員別の不良検出率を視覚化
- **詳細テーブル**: 検査員別の集計データ（検査数、見つけた不良、実際の不良など）

### 判定基準

| 平均検出率 | 判定 |
|----------|------|
| >= 95% | 良好（good） |
| >= 85% | 注意（warning） |
| < 85% | 警告（alert） |
